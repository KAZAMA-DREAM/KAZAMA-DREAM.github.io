JavaScript如何在HTML中引入

发表于2021-07-27|更新于2023-03-16
|字数总计:244|阅读时长:1分钟|阅读量:

三种引入方式

  • 1、在元素的事件属性中添加js代码，当事件触发时js执行

  • 2、内部： 在html页面中任意位置添加script（推荐head里面）

  • 3、外部： 在单独的js文件中写js代码，通过script的src属性引入，如果script标签引入了文件则不能在标签体内继续写js代码
