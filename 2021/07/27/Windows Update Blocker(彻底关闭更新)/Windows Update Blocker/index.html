软件 介绍

        以前的Windows10，可以自行在更新的设置里面关闭更新，但是现在的Windows10，只能暂停更新7天，要永久关闭更新，不仅要修改注册表，还要禁用服务，非常麻烦。这个小工具就是专门为阻止更新而开发的工具。

        软件自带多语言，支持命令行调用，一键关闭更新，非常方便。

        Windows Update Blocker是一个免费软件，可以帮助您通过单击按钮完全禁用或启用Windows系统上的自动更新。该实用程序是可移植的，这意味着您不再需要执行安装过程，并且在删除硬盘后，硬盘上不会残留任何残留物。如果将程序文件放置到外部数据设备上，则可以随身携带Windows Update Blocker，并在与您接触的任何计算机上使用它，该计算机的配置正确，并且不希望使用更新可能会破坏其系统的机会
