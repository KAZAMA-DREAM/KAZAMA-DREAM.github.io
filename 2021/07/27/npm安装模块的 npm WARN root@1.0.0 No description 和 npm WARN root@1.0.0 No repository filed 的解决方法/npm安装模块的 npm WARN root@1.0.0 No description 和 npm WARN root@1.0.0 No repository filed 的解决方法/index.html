执行

  • npm install vue –save

    出现下面两个错误

报错分析

  • 是因为系统在给你配置package.json文件的时候并没有帮你设置description字段和存储库字段，需要我们自己手动设置。可以用vim打开这个文件编辑，也可以打开源文件编辑


根据提示路径手动打开package.json文件修改 description和private字段

最后再运行

  • npm install vue –save

    完美解决
