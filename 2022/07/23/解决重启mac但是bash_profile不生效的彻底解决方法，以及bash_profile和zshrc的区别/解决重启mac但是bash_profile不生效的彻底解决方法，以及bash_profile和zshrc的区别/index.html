六、.bash_profile 和 .zshrc 的相关解释

1、符号解释
  • “~” ：你的home目录，在OS X下位于 /Users/你的用户名
  • “.” ：类unix下的隐藏文件，文件名带 “.” 之后在GUI文件管理器和ls的默认设置下不会显示出来，使用ls -a 命令可以显示出这些文件。zshrc是一个文件，准确说这个文件的文件名是”.zshrc”

2、zsh是什么？

zsh是shell语言类型，兼容bash，提供强大的命令行功能，比如Tab补全，自动纠错等。缺点就是配置太麻烦，好在有个叫做oh-my-zsh的开源项目，很好的弥补了这一缺陷，只需要修改配置文件即可
