Oracle 11g卸载教程

oracle11g卸载方法：首先找到oracle开头的所有服务，右击选择停止；然后找到Universal Installer，运行Oracle Universal Installer，单击卸载产品；最后重启电脑，删除oracle目录即可。

1、停用oracle服务：进入计算机管理，在服务中，找到oracle开头的所有服务，右击选择停止。

2、在开始菜单中，找到Universal Installer，运行Oracle Universal Installer，单击卸载产品。

3、在产品清单窗口中，单击全部展开，除了OraDb11g_home1外，勾选其他项目，单击删除,卸载完成后，再删除：OraDb11g_home1

4、按Windows键或CTRL+R键，打开运行窗口，输入regedit，打开注册表，依次展开HKEY_LOCAL_MACHINE\SOFTWARE，找到oracle，右键删除（该目录下注册着Oracle数据库的软件安装信息）

5、依次展开HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services中，删除该路径下所有oracle开头的项（这个是Oracle在windows下注册的各种服务）

6、依次展开HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Eventlog\Application，删除所有oracle开头项（这是注册的Oracle事件日志）

7、在HKEY_CLASSES_ROOT，删除以Ora、Oracle、Orcl或EnumOra开头的项。

8、删除HKEY_CURRENT_USER/SOFTWARE/Microsoft/windows/CurrentVersion/Explorer/MenuOrder/Start Menu/Programs中所有以oracle 开头的键。
9、删除HKDY_LOCAL_MACHINE/SOFTWARE/ODBC/ODBCINST.INI中除Microsoft ODBC for Oracle注册表键以外的所有含有Oracle的键。其中有些注册表项可能已经在卸载Oracle产品时被删除。
10、删除环境变量path中关于oracle的内容。（这一步好像不做也可以。）

鼠标右键右单击“我的电脑–>属性–>高级–>环境变量–>PATH 变量。

删除Oracle在该值中的内容。注意:path中记录着一堆操作系统的目录，在windows中各个目录之间使用分号（; ）隔开的，删除时注意。

建议：删除PATH环境变量中关于Oracle的值时，将该值全部拷贝到文本编辑器中，找到对应的Oracle的值，删除后，再拷贝修改的串，粘贴到PATH环境变量中，这样相对而言比较安全。

11、删除“开始”/“程序”中所有Oracle的组和图标。
12、重启电脑，删除oracle目录，删除Oracle的安装目录app。

包括：

1、c:\Program file\Oracle目录。

2、ORACLE_BASE目录。

3、c:\Documents and Settings\系统用户名、LocalSettings\Temp目录下的临时文件。
