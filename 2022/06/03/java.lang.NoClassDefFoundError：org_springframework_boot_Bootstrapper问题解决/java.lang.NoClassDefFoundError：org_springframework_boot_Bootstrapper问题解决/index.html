java.lang.NoClassDefFoundError: org/springframework/boot/Bootstrapper问题解决

1、问题分析：

Spring Boot 依赖版本和Spring Cloud Alibaba依赖版本不一致，导致报错。
解决办法：根据以下信息将Spring Boot 依赖版本和Spring Cloud Alibaba依赖版本一一对应。

2、官方文档

版本说明 · alibaba/spring-cloud-alibaba Wiki · GitHub
