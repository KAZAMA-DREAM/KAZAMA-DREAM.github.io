Nacos Server did not start because dumpservice bean construction failure : No DataSource set异常解决

1、第一种情况：nacos和mysql都部署在本地机器

【1】先检查mysql服务是否启动，如果没有启动，就启动mysql后再尝试启动nacos（此时mysql中一定要已经导入了nacos-mysql.sql文件初始化了数据库了）

【2】如果mysql已经启动了nacos启动还是报错，修改connectTimeout、socketTimeout

最后，修改了数据库url配置参数，解决了问题：

&connectTimeout=10000&socketTimeout=30000

原配置是&connectTimeout=1000&socketTimeout=3000

其他网友解答：

加了&serverTimezone=UTC还是不行，加上&allowPublicKeyRetrieval=true就好了

我跟你们说正解，你用其他的客户端先连一下数据库 随便查询点啥，唤醒他
